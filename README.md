# heart-disease-prediction-

 Key Steps :
 
Data Cleaning & Preprocessing
Handled missing values, encoded categorical data, scaled numerical features.

Exploratory Data Analysis (EDA) :
Identified patterns and key risk factors using visualizations.

Feature Engineering & Selection :
Selected top features using SelectKBest (Chi-Square test).

Model Training :
Trained Logistic Regression, Decision Tree, and Random Forest classifiers.

Hyperparameter Tuning :
Used GridSearchCV for tuning Random Forest parameters.

Model Evaluation :
Metrics: Accuracy, Precision, Recall, F1 Score, ROC-AUC.

 Results
Best Model: Random Forest Classifier
Best Accuracy: 98% 
ROC-AUC Score: 98


 Real-World Use Case
This model can assist in early detection of heart disease, enabling timely medical intervention. It can be integrated into health apps or used by clinics for risk assessment.


 Tools & Technologies
Language: Python
Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
Environment: Jupyter Notebook / Google Colab
